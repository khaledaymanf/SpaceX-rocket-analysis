# SpaceX Rocket Fleet Analysis

## Overview
Analysis of SpaceX rockets using Python, Pandas, Plotly, and Scikit-learn.
Performed data cleaning, visualization, and basic machine learning models.

## Skills
- Data Wrangling
- Exploratory Data Analysis
- Visualization (Matplotlib, Seaborn, Plotly)
- Machine Learning (Decision Tree, SVM, KNN)

## Summary
This project provides insights on cost efficiency, payload capacity, and strategic recommendations for a hypothetical startup "Space Y".
